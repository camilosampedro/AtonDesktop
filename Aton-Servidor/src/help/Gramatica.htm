Gramática

Una gramática es un conjunto de normas que definen la construcción de un lenguaje. Por ejemplo:

  1. <frase> -> <sujeto> <verbo> <predicado>
  2. <sujeto> -> <articulo> <sustantivo>
  3. <articulo> -> el
  4. <articulo> -> la
  5. <sustantivo> -> niño
  6. <sustantivo> -> quiere
  7. <predicado> -> <articulo> <sustantivo>
